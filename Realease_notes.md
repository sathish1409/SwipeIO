
# SwipeIO v1.1.0
 
## Bug Fixes:

1. Taking the log for calculation, only if the **emp_id** matches with the **card_id**.

2. Improved performance.

  

## Updates:

1. Incharges and Admins can regularize the logs of their reporting employees.

2. If the log is regularized, then the report and log will be shown with the speacial label **R**.

3. Search field is provided in the employees listing page.(Employee can searched by their Id, Name and Email).

4. Search field is provided in the 'select employee' drop down in the report page.

5. Show password is provided in employee add and edit page.

6. Note page is now also provided for reporting employees.
